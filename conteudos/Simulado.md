## Questões de IA, ML e AWS

---

## 1. Tipo de dados utilizado

**Pergunta:**  
Uma empresa está prevendo as vendas do próximo ano com base em dados de vendas mensais dos dois anos anteriores. O conjunto de dados está em formato CSV e consiste no ano, mês e o total de vendas correspondente a cada mês.

**Alternativas:**  
- A. Dados estruturados  
- B. Dados não estruturados  
- C. Dados semiestruturados  
- D. Dados de texto  

**Resposta:**  
**A. Dados estruturados**

---

## 2. Métrica para avaliar a qualidade de resumos

**Pergunta:**  
Uma empresa financeira deseja avaliar um modelo de linguagem de grande escala (LLM) usado para sumarização de textos. A empresa quer avaliar a qualidade dos resumos.

**Alternativas:**  
- A. BLEU  
- B. Precisão média (mAP)  
- C. ROUGE  
- D. Pontuação F1  

**Resposta:**  
**C. ROUGE**

## 3. Limpeza e transformação de dados sem código

**Pergunta:**  
Um profissional de IA está criando um modelo de previsão para estimar preços de ações. As fontes de dados analisadas contêm valores ausentes e outliers estatísticos. O profissional de IA deseja limpar e transformar os dados sem escrever código personalizado.

**Alternativas:**  
- A. Tarefas do AWS Glue  
- B. Amazon SageMaker Data Wrangler  
- C. Amazon Athena  
- D. Loja de recursos do Amazon SageMaker  

**Resposta:**  
**B. Amazon SageMaker Data Wrangler**

## 4. Responsabilidade compartilhada - segurança

De acordo com o modelo de responsabilidade compartilhada da AWS, qual opção é de responsabilidade exclusiva do cliente?

## 7. Inferência quase em tempo real

**Pergunta:**  
Um profissional de IA está testando um modelo hospedado no Amazon Bedrock. Ele configurou o modelo para exigir apenas o primeiro token retornado. O restante da resposta é sempre desnecessário.  

**Alternativas:**  
- A. Defina o comprimento da resposta como 1  
- B. Ajuste a temperatura para 1  
- C. Defina o valor Top P como 1  
- D. Defina o Top K como 1  

**Resposta:**  
**A. Defina o comprimento da resposta como 1**
